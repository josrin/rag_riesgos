# ─── RAG Riesgos — Dependencias Python ───
# Probado con Python 3.13.12 en Windows 11
# Instalar dentro del venv: pip install -r requirements.txt
#
# PREREQUISITOS EXTERNOS (no se instalan con pip):
#   1. Ollama >= 0.20     → https://ollama.com/download/windows
#      Modelos requeridos (descargar con `ollama pull <nombre>`):
#        - llama3              (LLM, ~4.7 GB)
#        - bge-m3              (embeddings multilingues, ~1.2 GB)
#   2. Tesseract-OCR >= 5  → https://github.com/UB-Mannheim/tesseract/wiki
#      Instalar con idioma "Spanish" habilitado.
#      Ruta por defecto: C:\Program Files\Tesseract-OCR\tesseract.exe
#   3. Poppler for Windows → https://github.com/oschwartz10612/poppler-windows/releases
#      Descomprimir y anotar la ruta a la carpeta bin.
#      Ej: C:\Program Files\poppler-24.08.0\Library\bin
#
# Las rutas de Tesseract y Poppler se configuran en .env
# (ver .env.example como plantilla).
